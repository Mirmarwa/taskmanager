# Application Web de Gestion de Tâches Collaborative

## Description
Cette application permet de gérer des projets et des tâches de manière collaborative.
Les utilisateurs peuvent créer des espaces de travail, ajouter des projets et organiser
les tâches selon leur avancement.

## Fonctionnalités principales
- Authentification (inscription / connexion)
- Gestion des workspaces
- Gestion des projets
- Gestion des tâches
- Tableau de bord avec statistiques

## Technologies utilisées
- Symfony (PHP)
- HTML / CSS / JavaScript
- MySQL
- Doctrine ORM

## Installation
1. Cloner le projet
2. Configurer le fichier .env (base de données)
3. Lancer les migrations
4. Lancer le serveur Symfony
